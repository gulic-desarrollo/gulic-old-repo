Rabiche LIVE

Con la colaboración del secretariado del software libre de la ULL, este año también hemos regalado una distribución de Linux a los asistentes.

Se trata de una modificación de una knoppix 4 en español, añadiendo los programas que se han visto en el curso que no estaban en el cd. La knoppix es una distribución LIVE que se ejecuta desde cdrom sin necesidad de instalarla en el ordenador, ni de realizar particiones. Próximamente pondremos URL de descarga para los que no asistieron al curso.

La página oficial de KNOPPIX en español es http://www.knoppix-es.org/. Hay secciones de foros donde poder leer mucho.

Tenemos tres formas de proceder con este live-cd, que pasamos a detallar, en orden creciente de dificultad:

  1. Ejecución como live-cd directamente desde el cdrom: nivel de dificultad nulo

    Solamente tenemos que configurar nuestro ordenador para que arranque desde cdrom y meter el cdrom y encender el ordenador. Es la forma más sencilla y rápida de probar Linux y 2GB de aplicaciones en tu ordenador. Eso sí, al ser ejecutada desde cdrom, la velocidad no es excesivamente rápida.

    Si metes el cdrom, enciendes el ordenador y arranca windows como siempre, es que tu bios no está configurada correctamente. Para hacer más suave el trance de entrar y configurar la bios, hemos creado unas instrucciones de configuración de bios con capturas de pantalla e instrucciones detalladas para bios de diferentes ordenadores. Hemos publicado varias, pero hay muchas bios diferentes. Si alguna es parecida a la tuya, no deberías tener problema. Si no encuentras ninguna parecida, leyéndote el resto de bios verás que no tiene mucha ciencia.

    Hay una desventaja de este método y es que cada vez que iniciamos del cdrom perdemos nuestras configuraciones personales que hagamos desde linux, pues al no usar ningun disco duro no se almacenan. Esto se soluciona creando una imagen de home persistente.

  2. Instalación en windows:

    Hay un método intermedio que, sin instalarla, conseguimos un linux como el del cd-live, pero sin estar teniendo que meter el cd y tal. Tiene el mismo problema que el cd-live de que perdemos las cosas que hagamos, y se soluciona de la misma forma, creando el home persistente.

    Básicamente el método es copiar el cdrom de linux a unas carpetas concretas en windows. Luego, para arrancar, hay dos alternativas:

    • Usando un diskette: nivel de dificultad bajo

      Si metemos el diskette, arrancamos linux... si lo quitamos, arrancamos windows. Esto no está documentado aún, lo estará en los próximos días.

    • Modificando el arranque de windows para generar un menú nuevo: nivel de dificultad medio-bajo

      Las instrucciones están en el siguiente enlace: Instalación PMI de Knoppix sin reparticionar

  3. Instalación tradicional: nivel de dificultad medio-alto

    En esta página en inglés [Hd install howto] aparece el método de instalación tradicional, que es el que aporta todas las ventajas de Linux, pero requiere cierta "habilidad". Consta (resumiendo) de los siguientes fases:

    • Redimensionamiento de la partición actual de Windows con algún software al efecto (p.ej: partition magic), para dejar espacio libre al final del disco duro
    • Arrancar la knoppix
    • Ejecutar el instalador
      • Crear las particiones de linux: asignar particiones para linux en el espacio libre con QTParted
      • Configurar las opciones del instalador: estilo de instalación, particion de destino, contraseña de root, usuario normal, etc
      • Ejecutar la instalación

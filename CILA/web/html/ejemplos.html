Ejemplos de código

Debido a la corta duración de algunas ediciones del CILA, hemos preparado unos ejemplos de código para mostrar

Tras terminar el CILA 2001 congelamos el desarrollo de los apuntes, y más adelante nos dimos cuenta de que el formato SGML no sería útil cuando el documento creciera. La versión final de estos apuntes está disponible aquí para bajarla en varios formatos:

  • SGML: empaquetados con tar un comprimido con gzip o bien comprimidos con zip
  • HTML: navegable, empaquetados con tar un comprimido con gzip o bien comprimidos con zip
  • PostScript: comprimidos con gzip o bien comprimidos con zip
  • PDF: sin comprimir

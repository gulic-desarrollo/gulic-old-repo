Ejemplos de código

Debido a la corta duración de algunas ediciones del CILA, hemos preparado unos ejemplos de código para ahorrar tiempo a la hora de mostrar a los alumnos el uso de herramientas de programación o formatos de documentación. Estos ejemplos de código están también incluidos en las páginas de los apuntes y del libro.

Los ejemplos de código están disponibles aquí para descargarlos empaquetados con tar y comprimidos con gzip o bien comprimidos con zip.
